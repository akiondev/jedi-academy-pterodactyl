# Support

## Getting help

Use the support channel that best matches your problem.

## Project support

Use this repository first for issues related to:

- the Docker image
- the Pterodactyl egg
- startup and runtime behavior
- addon loading and managed helpers
- documentation
- anti-VPN integration

Useful project links:

- [README.md](README.md)
- [CONTRIBUTING.md](CONTRIBUTING.md)
- [SECURITY.md](SECURITY.md)
- [Report Bug](https://github.com/akiondev/jedi-academy-pterodactyl/issues/new?title=BUG:%20)
- [Request Feature](https://github.com/akiondev/jedi-academy-pterodactyl/issues/new?title=REQUEST:%20)

For quick questions, you can also contact:

- Discord: `akion`

GitHub is still preferred for bugs, feature requests, and anything that should remain searchable or trackable.

## Official Pterodactyl resources

If your problem is really about Pterodactyl itself, these are the most useful official references:

- [Pterodactyl website](https://pterodactyl.io/)
- [Pterodactyl documentation](https://pterodactyl.io/project/introduction.html)
- [Pterodactyl panel repository](https://github.com/pterodactyl/panel)
- [Pterodactyl Wings repository](https://github.com/pterodactyl/wings)
- [Pterodactyl yolks repository](https://github.com/pterodactyl/yolks)

## Community and unofficial Pterodactyl resources

These can still be useful, but they are not the official project:

- [Ptero-Eggs community organization](https://github.com/ptero-eggs)
- [Pterodactyl Installer (unofficial)](https://github.com/pterodactyl-installer/pterodactyl-installer)

## TaystJK, OpenJK, and Jedi Academy references

## Docker and host references

Since Pterodactyl runs servers inside containers, Docker can also be relevant:

- [Docker documentation](https://docs.docker.com/)

For Linux host issues, use the documentation for your own distribution and hosting environment.

## Reporting security issues

For security-sensitive problems, do not open a public issue first.

Please follow [SECURITY.md](SECURITY.md).
