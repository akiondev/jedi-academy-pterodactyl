# Bundled example announcer messages
# Blank lines and lines starting with # are ignored.
#
# By default the worker rotates through these lines once every
# `interval_seconds` (see 20-python-announcer.config.json).
#
# To send messages at exact times of day instead, populate the `schedule`
# list in the config. Each entry is one of:
#
#   {"time": "HH:MM"}                      daily at HH:MM (in `timezone`)
#   {"time": "HH:MM:SS"}                   second-precision daily slot
#   {"every_seconds": 1800}                fires every N seconds (>= 30)
#
# Any entry may add a literal "message": "..." to override the rotation
# for that slot. Example:
#
#   "schedule": [
#     {"time": "12:00", "message": "Lunch break! Come back at 13:00."},
#     {"time": "20:00"},
#     {"every_seconds": 1800}
#   ]
#
# `announce_command` accepts "svsay" (default, server message) or "say"
# (speaks as the dedicated console). Any other value is rejected.

Welcome to the bundled TaystJK example announcer.
Edit 20-python-announcer.messages.txt to change these recurring server messages.
Set SERVER_RCON_PASSWORD in the egg or rconpassword in the active server config to enable bundled RCON announcements.
Rename or remove 20-python-announcer.py if you do not want to run the bundled announcer.
