# v0.1.0 (alpha)

First public snapshot of the TaystJK Pterodactyl project.

This was the initial proof-of-concept release. The project was in an
**alpha** state — functional enough to run a server, but with no
stability guarantees for the egg format, startup variables, or image
layout.

## Included

- Source-built Docker image for TaystJK dedicated server
- Importable Pterodactyl egg
- Runtime entrypoint script
- Installation helper script
- Source-analysis and revision notes

This project does **not** redistribute copyrighted Jedi Academy game
assets. Users must provide their own legally owned files.
